# Как добавить мемы в бота

## 🚀 Самый простой способ (рекомендуется)

1. **Запустите бота:**
   ```bash
   python bot.py
   ```

2. **В Telegram отправьте боту команды для каждого мема:**
   - Отправьте `/add_meme`
   - Введите название мема (например: "Объект нагревается - его атомы")
   - Отправьте фото мема
   - Повторите для каждого мема

**Названия мемов:**
1. "Объект нагревается - его атомы"
2. "Спасибо сломанной вытяжке - дымкор эстетик"
3. "14 000 625 измерений - один сошёлся"
4. "Неупругая деформация"

---

## 🤖 Автоматический способ (если у вас есть файлы)

1. **Создайте папку `memes`** в директории бота

2. **Поместите изображения мемов** в папку `memes`:
   - Любые названия файлов (.jpg, .png и т.д.)
   - Скрипт автоматически найдёт все изображения

3. **Запустите скрипт:**
   ```bash
   python upload_memes.py
   ```

4. **Готово!** Мемы автоматически загрузятся в бота и добавятся в базу данных.

---

## 📋 Альтернативный способ (через file_id)

1. **Запустите бота**

2. **Для каждого изображения:**
   - Отправьте `/get_file_id`
   - Отправьте фото
   - Скопируйте полученный file_id

3. **Используйте скрипт `add_memes.py`:**
   - Вставьте file_id в скрипт
   - Запустите: `python add_memes.py`

---

## ✅ Проверка

После добавления мемов:
- Откройте бота в Telegram
- Нажмите кнопку "😂 Мемы"
- Должны появиться кнопки навигации "◀️ Предыдущий мем" и "Следующий мем ▶️"
