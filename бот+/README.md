# Simple Physics Telegram Bot 🤖

Простой бот для проекта Simple Physics.
Всё в одном файле для простоты!

## 🚀 Быстрый старт

### 1. Установите зависимости

```bash
pip install python-telegram-bot
```

### 2. Найдите свой ID

1. Откройте [@userinfobot](https://t.me/userinfobot) в Telegram
2. Отправьте ему `/start`
3. Скопируйте ваш ID (число)

### 3. Откройте bot.py и замените

```python
ADMIN_ID = 0  # ЗАМЕНИТЕ НА СВОЙ ID!
```

На ваш ID:
```python
ADMIN_ID = 123456789  # ваш реальный ID
```

### 4. Запустите бота

```bash
python bot.py
```

Готово! Бот работает! 🎉

## 📖 Как использовать

### Для пользователей

1. Найдите вашего бота в Telegram
2. Отправьте `/start`
3. Выберите раздел из меню

### Для администратора

**Добавить комикс:**
```
/add_comic
Введите название → Отправьте фото
```

**Добавить карточку:**
```
/add_card
Введите название → Отправьте фото
```

**Добавить видео:**
```
/add_video
Введите название → Отправьте ссылку
```

## 📁 Файлы проекта

- `bot.py` - весь код бота в одном файле
- `physics_bot.db` - база данных (создаётся автоматически)
- `requirements.txt` - список библиотек

## 💡 Что делает бот?

- Показывает комиксы по физике
- Показывает исторические карточки
- Показывает видео с опытами
- Информация о проекте
